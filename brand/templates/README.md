# Template reference

Master assets for reuse. Match these specs when creating new posts in Canva/Figma.

## Canvas

| Template | Size | Aspect |
|----------|------|--------|
| Feed post | 1080 × 1080 | 1:1 |
| Portrait feed | 1080 × 1350 | 4:5 |
| Story | 1080 × 1920 | 9:16 |
| Profile | 1080 × 1080 | 1:1 |
| Facebook cover | 1640 × 624 | ~2.6:1 |

## Layout grid (1080 feed)

- **Safe margin:** 80px all sides
- **Logo:** top-left, 64px icon or 120px lockup
- **Headline:** y ≈ 340–420, max 2 lines, 72–96px
- **Subcopy:** y ≈ 520–680, 32–40px, max 3 lines
- **Footer URL:** bottom 80px, centred or left, 28px, colour `#db2777`
- **Glow accent:** soft radial gradient behind headline, pink at 15–25% opacity

## Reusable slide types

1. **Stat card** — one big number or phrase (post 01, 02)
2. **Feature spotlight** — headline + 2-line benefit (post 04, 07, 08)
3. **Comparison** — two columns with ✓ and ✗ (post 05)
4. **Product hero** — phone mockup + headline (post 06)
5. **Steps** — numbered 1-2-3 (post 09)
6. **Launch CTA** — logo + tagline + URL (post 10)
